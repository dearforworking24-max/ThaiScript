# 🇹🇭 Thai Script

A custom programming language built entirely from scratch for educational purposes. This project is designed to explore and demonstrate the fundamental operations of programming languages, rather than just translating existing code.

## 💡 Project Overview
The goal of this project is to understand the core concepts behind interpreters/compilers, including lexical analysis (Lexer), parsing, and evaluation. Thai Script features its own unique syntax rules and logic structure.

## ✨ Features (so far)
- Custom syntax design and parsing.
- Function definitions and executions.
- *(เพิ่มฟีเจอร์ที่คุณทำได้แล้ว เช่น Basic arithmetic operations, Variable declarations)*
